# Homebrew Tap for Coni

This is the official [Homebrew](https://brew.sh/) tap for [Coni](https://github.com/coni-ai/coni).

## Installation

```bash
brew tap coni-ai/coni
brew install --cask coni
```

## Upgrade

```bash
brew upgrade --cask coni
```

## Uninstall

```bash
brew uninstall --cask coni
```
